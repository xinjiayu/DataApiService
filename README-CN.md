#DataApiService

[English](./README.md) | 简体中文


MySql数据库的API服务

支持数据查询，采用json输出。

支持基于JWT的用户验证


##安装

DataApiService代理是跨平台的，它支持Linux、Unix、Mac OS和Windows。DataApiService可以部署在x86_64架构服务器和ARM设备上。

##源码编译

```bash
git clone https://github.com/xinjiayu/DataApiService.git

cd DataApiService

go build

./DataApiService

```
